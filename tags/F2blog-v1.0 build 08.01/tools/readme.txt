TT转F2blog使用说明：
    1. tttof2blog.php：用于把TT1.0以上版本的数据转成f2blog专用格式。
    2. tt96f2blog.php：用于把TT0.96版本的数据转成f2blog专用格式。
    3. f2blog.php：　　用于把f2blog专用格式的数据汇入到f2blog中。
    4. 上传tools目录到服务器下，如果是linux或unix服务器，请确保tools目录属性是777，接着运行
       tools/tttof2blog.php或者tools/tt96f2blog.php文件，按说明提示输入连接TT的mysql资料。
       同时可以选择要汇出的资料等，如果数据非常大，最好分表来汇出。
    5. 汇出成功后，可以备份好tt_data.dat文件。以备下次使用，默认情况直接下一步就可以进入f2blog
    　 通用程序的汇入工作。
    6. 输入连接f2blog的mysql的资料，选择要汇入的数据，然后下一步就可以完成了，汇入完成后，需要
    　 登入后台重新建立缓存。
